## 🎯 核心知识点速览

### 1. 风险与回报 (Risk and Return)
游戏乐趣的核心在于"风险与回报"的平衡：
- 高风险高回报带来满足感
- 让玩家能够预判风险和回报
- 避免"只有风险没有回报"

### 2. 难度曲线设计
- 单调递增的难度会让玩家疲劳
- 应该有"简单→困难→简单"的节奏
- 给玩家"休息区"和"挑战区"

### 3. 企划书写作
- 核心企划书应该能写在一页纸上
- 明确5W1H：Who、What、When、Where、Why、How
- 概念先行，先确定核心乐趣

### 4. 职业素养
- 游戏开发是工作，不是娱乐
- "总之先去做"的精神
- 输入输出都要大量
- 苦劳会被遗忘，作品会留下

### 5. 帧数的重要性
- **60fps = 16.67ms/帧**
- 玩家输入后应在3帧内（约50ms）有反馈
- 攻击动作要有预备动作

### ✅ A-职业素养（14话）
涵盖主题：
- 游戏开发绝非游戏
- 演讲就要多快好省
- 企划书的写法
- 开发期内的需求变更
- 日报的建议
- 苦劳会忘记，作品会留下
- 总之先去做!!
- 不要依赖备份方案
- 概念要贯彻始终
- 我如何坚持编写《Fami通》专栏
- 定位目标受众
- 输入输出都要大量
- 事成之前切忌声张

### ✅ B-游戏性（22话）
涵盖主题：
- 风险与回报理论
- 不同类型游戏的游戏性（射击、格斗、动作、RPG、益智、赛车等）
- 有峰有谷的难度曲线
- 比无法通关要好
- 重试要迅速
- 发扬长处，也补长短处
- 奖励要素要优先
- 敌人是否真的必要
- 游戏性提高则一般性下降
- 游戏是什么？

### ✅ C-企划设计（20话）
涵盖主题：
- 帧率的重要性
- 企划书的写法
- 如何靠做游戏谋生
- 要会计算帧数
- 一键式游戏
- 分解、考察、再构筑
- 创新性需要到什么程度
- 教程要自然
- 目标定位与营销
- 对新手友好、舒适
- 意识到占用的时间
- 避免用现有作品参照

## 💡 学习方法建议

1. **按分类学习**：选择自己感兴趣或需要提升的领域深入学习
2. **结合实际**：观看视频时同步打开相关游戏进行对照
3. **做笔记**：每话都有核心知识点，建议用自己的话总结
4. **实践应用**：尝试将学到的知识应用到自己开发的游戏中
